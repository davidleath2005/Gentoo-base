# Gentoo Base Installation Guide

A comprehensive, copy-paste-friendly Gentoo Linux base installation guide for AMD64 systems.

- **Boot method:** EFI Stub (no GRUB)
- **Init system:** OpenRC
- **Kernel:** Distribution kernel (built from source, Gentoo config)
- **Target disk:** NVMe
- **fstab:** UUID-based (critical for multi-drive setups)

See [INSTALL_GUIDE.md](INSTALL_GUIDE.md) for the full step-by-step guide.
